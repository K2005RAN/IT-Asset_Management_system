# 📖 Overview

Industrial organizations often rely on spreadsheets to monitor IT assets, resulting in fragmented data, inconsistent tracking, delayed maintenance, and poor visibility into asset utilization.

This project digitizes the complete asset lifecycle by providing a centralized platform capable of:

- Asset Registration
- Employee Allocation
- Preventive Maintenance
- Repair Workflow
- Warranty Tracking
- Vendor Management
- Location Tracking
- Real-Time Dashboard
- Complete Audit History

The system enables organizations to manage thousands of IT assets efficiently from a single dashboard.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/K2005RAN/IT-Asset_Management_system.git

cd IT-Asset_Management_system
```

---

# 📦 Backend Setup

```bash
cd server

npm install
```

Create a `.env`

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret

EMAIL=your_email

PASSWORD=your_password
```

Run Backend

```bash
npm start
```

---

# 💻 Frontend Setup

```bash
cd ui

npm install

npm run dev
```

---

# 🐳 Docker Deployment

Run the complete application using Docker Compose.

```bash
docker-compose up --build
```
