# Industrial IT Asset & Lifecycle Management System

A robust, full-stack **MERN platform** designed to orchestrate, audit, and track the operational lifecycle of industrial IT hardware infrastructure across distributed plant locations. 

This system bridges the gap between spreadsheet-based tracking and automated enterprise logistics, providing real-time data synchronizations, an interactive workflow engine for maintenance logistics, and structured database models.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS, Vite runtime engine, Lucide Icons.
* **Backend:** Node.js, Express.js (RESTful API architecture layer).
* **Database:** MongoDB mapped via Mongoose Object Data Modeling (ODM).

---

## 📦 Project Structure

This repository contains the core application source code:
* **`server/`**: The Node.js and Express.js API application service layer, database schemas, and routing logic.
* **`ui/`**: The React.js single-page application (SPA) client user interface.
